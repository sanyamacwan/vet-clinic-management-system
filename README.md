# Veterinary Clinic Management System



This repository contains an academic \*\*C# application\*\* designed to manage core operations of a veterinary clinic.



The project demonstrates object-oriented programming, structured system design, and backend logic for managing entities such as patients, appointments, and clinic data. It was developed as part of my academic coursework and is shared here for portfolio and learning purposes.



## Features

\- Clinic and patient management

\- Structured backend logic

\- Clean separation of concerns

\- Academic system design and implementation



## Technologies Used

\- C#

\- .NET / ASP.NET (depending on configuration)

\- Visual Studio



## Academic Context

This project was completed as part of my academic studies.
